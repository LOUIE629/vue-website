# 基于Vue.js框架制作的视频网站视频播放量、用户量统计分析网站
本系统，前端使用Vue3框架，使用element-plus、echarts、vuex、vue-router库；后端使用Express框架，使用MongoDB数据库(Mongoose管理)、nodemon库。使用Axios进行通信，调用了百度地图API以获取用户位置信息。

注意！！！

须在本地安装MongoDB数据库

具体环境配置查看文件夹中的readme
